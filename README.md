# РГР: Разработка Web-приложения для инференса моделей ML и анализа данных

## О проекте

Данный проект представляет собой веб-приложение (дашборд), разработанное на **Streamlit**, предназначенное для:
- Инференса (вывода) моделей машинного обучения
- Визуализации данных о продажах недвижимости
- Анализа зависимостей между признаками

**Датасет:** House Sales in King County, USA (продажи жилой недвижимости в округе Кинг, штат Вашингтон, США за 2014-2015 гг.)

**Целевая переменная:** `price` - цена продажи дома

## Модели машинного обучения

В проекте реализованы 6 моделей для задачи регрессии (предсказание цены дома):

| № | Модель | Тип | Библиотека |
|---|--------|-----|------------|
| 1 | Полиномиальная регрессия | Классическая | scikit-learn |
| 2 | Gradient Boosting | Ансамблевая (бустинг) | scikit-learn |
| 3 | CatBoost | Продвинутый градиентный бустинг | catboost |
| 4 | Bagging | Ансамблевая (бэггинг) | scikit-learn |
| 5 | Stacking | Ансамблевая (стэкинг) | scikit-learn |
| 6 | Полносвязная нейронная сеть | Глубокая сеть | TensorFlow/Keras |

## Страницы приложения

### 1. О разработчике
- ФИО, номер группы, фотография
- Тема РГР
- Ссылки на GitHub и Streamlit Cloud

### 2. О датасете
- Описание предметной области
- Список признаков с пояснениями
- Шаги предобработки данных (удаление дубликатов, оптимизация типов, создание новых признаков)
- Результаты EDA

### 3. Визуализации
- 4 графика:
  - Распределение цены недвижимости
  - Зависимость цены от жилой площади
  - Влияние качества дома на цену
  - Корреляционная матрица признаков

### 4. Инференс (предсказание)
- Выбор модели из 6 доступных
- Форма для ввода характеристик дома (более 20 признаков)
- Валидация ввода (границы из датасета)
- Вывод предсказанной стоимости в долларах
